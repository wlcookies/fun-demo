### 分辨率相关

```bash

# 查看设备分辨率（px）
adb shell wm size

# 查像素密度（dpi）
adb shell wm density

# 修改分辨率（px）中间是x不是*
adb shell wm size 800x800

# 重置分辨率
adb shell wm size reset

```
